接口说明

搜索歌词并返回最佳匹配，供第三方应用集成调用。

请求地址 
{plugin_base}/open-scrape
请求方法 
POST
请求头 
Content-Type: application/json
Authorization: Bearer <token>
请求参数 (JSON Body)
参数 类型 必填 说明
titlestring是歌曲名
artiststring否歌手名
durationnumber否时长（毫秒），用于提高匹配准确度
响应参数
参数 类型 说明
successboolean是否成功
sourcestring来源标识（qm/kg/ne/lrclib）
source_labelstring来源显示名（小秋/小枸/小芸/Lrclib）
scorenumber匹配度（0-100）
match_titlestring匹配到的歌曲名
match_artiststring匹配到的歌手名
match_durationnumber匹配到的歌曲时长（毫秒）
lyrics_typestring歌词类型：verbatim（逐字）/ line（逐行）
line_countnumber歌词行数
lyricstring原始 LRC 歌词文本
tlyricstring翻译 LRC（可能为空）
lxlyricstring逐字时间轴 LRC（可能为空）
响应示例 (成功) 
{
  "success": true,
  "source": "qm",
  "source_label": "小秋",
  "score": 92,
  "match_title": "七里香",
  "match_artist": "周杰伦",
  "match_duration": 245000,
  "lyrics_type": "verbatim",
  "line_count": 42,
  "lyric": "[00:03.35]词：天天\n[00:06.70]曲：周杰伦\n[00:16.76]七月的风懒懒的...",
  "tlyric": "",
  "lxlyric": "[00:00.00][[00:00.00]]七[[00:00.18]]月..."
}
响应示例 (失败) 
{
  "success": false,
  "message": "未找到匹配的歌词"
}
